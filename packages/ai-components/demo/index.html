AI 驱动的
AI Components 组件库

用自然语言驱动的 Web Component 组件库。每个组件接收自然语言 prompt，调用 LLM 流式生成并实时渲染内容。 纯 Web Component 实现，可在 React / Vue / Svelte / 原生 HTML 中使用。

组件之间可以互相引用和嵌套——LLM 生成的 HTML 中可以包含其他 AI 组件标签，甚至 <ai-component> 可以递归调用自己，实现分形式的 AI 生成。

快速开始 → 🎮 在线体验

核心特性

🧩
增量 DOM 渲染
不使用 innerHTML 替换，通过 createElement / appendChild 逐步构建 DOM，避免闪烁和引用断裂。
🌊
流式缓冲解析
跨 chunk 边界的不完整 HTML 标签会被缓冲，等待后续数据完成后再处理，保证解析正确性。
🔄
递归嵌套 & 深度感知
LLM 输出中的 AI 组件标签自动触发新生成。支持最多 3 层嵌套，每层有独立的提示词策略。
🔌
框架无关
纯 Web Component 实现，可在 React、Vue、Svelte、Angular 或原生 HTML 中使用，零适配成本。
✨
零运行时依赖
完全无第三方依赖。IIFE 压缩后仅 ~8 KB，支持 tree-shaking 的 ESM 产物。
🛡️
完整的错误处理
每个组件独立捕获错误，不影响父子组件。支持 AbortController 中断和友好的错误展示。

⚡ 快速上手

1. 安装
# 选择你的包管理器
npm install ai-components
pnpm add ai-components
2. 使用
<ai-component
  p="生成一个用户注册表单"
></ai-component>

组件一览

🧩 <ai-component> 可用 核心

通用 AI 组件。接受任意自然语言 prompt，流式生成并渲染 HTML 片段。支持递归嵌套和深度感知。
📊 <ai-data> 可用 基础

数据提供组件。通过 LLM 异步获取数据，配合任意 AI 组件使用。支持嵌套、不渲染内容。
✏️ <ai-text> 可用

AI 纯文本组件。流式输出纯文本内容，不解析 HTML。适合标题、段落、文案等场景。
📝 <ai-markdown> 可用

AI Markdown 组件。LLM 生成 Markdown 并自动渲染为精美 HTML。内置多种主题。
🖼️ <ai-image> 可用

AI 图片搜索组件。通过自然语言描述图片内容，自动搜索最匹配的免费高清图片并展示。
🎨 <ai-canvas> 可用

AI 文生图组件。通过自然语言描述，调用 DALL-E 兼容 API 生成图片。支持标准 img 属性。
🗺️ <ai-map> 可用

AI 地图组件。自然语言描述地点或路线，自动解析坐标并展示交互式 OpenStreetMap 地图。
🎵 <ai-music> 可用

AI 音乐播放组件。自然语言描述想听的音乐，自动搜索并播放。支持 LLM 定制播放器 UI。
