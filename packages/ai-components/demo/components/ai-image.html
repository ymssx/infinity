<ai-image>

可用

AI 驱动的智能图片搜索组件——通过自然语言描述图片内容，自动搜索最匹配的图片并展示。 支持标准 <img> 标签的所有属性（width、height、alt、loading 等）。

⚙️ 属性（Attributes）

属性类型默认值说明
p string — 必填。自然语言描述想要的图片内容。修改后自动重新搜索。
width number 400 图片宽度（像素）
height number 300 图片高度（像素）
alt string 同 p 属性 图片替代文本，默认使用 p 属性的值
loading string — 懒加载策略：lazy 或 eager

💡 使用示例

基本用法

<ai-image p="一只在雪地里奔跑的柯基"></ai-image>

指定尺寸

<ai-image
  p="夕阳下的海边灯塔"
  width="800"
  height="400"
  loading="lazy"
></ai-image>

高级配置（使用 Unsplash API Key）

<script>
  // 可选：配置 Unsplash API Key 获得更精准的搜索结果
  window.__ai_image_config = {
    unsplashAccessKey: 'your-unsplash-access-key'
  };
</script>

<ai-image p="minimalist workspace with coffee and laptop"></ai-image>

🔍 工作原理

1. 解析描述
读取 p 属性中的自然语言描述作为搜索关键词。
2. 搜索图片
使用 Unsplash API 搜索最匹配的免费高清图片。无需 API Key 即可使用基础功能。
3. 渲染展示
图片加载完成后淡入显示，支持所有标准 <img> 属性。
