<ai-canvas>

可用

AI 驱动的文生图组件——通过自然语言描述想要的图片，调用文生图 API（如 DALL-E、Stable Diffusion）生成图片。 支持标准 <img> 标签的属性。

⚙️ 属性（Attributes）

属性类型默认值说明
p string — 必填。自然语言描述想要生成的图片。修改后重新生成。
width number 512 图片显示宽度（像素）
height number 512 图片显示高度（像素）
alt string 同 p 属性 图片替代文本

🔧 配置

使用前需要配置文生图 API。支持所有 OpenAI Images API 兼容的服务。 

<script>
  // 配置文生图 API
  window.__ai_canvas_config = {
    apiUrl: 'https://api.openai.com/v1/images/generations',
    apiKey: 'sk-your-api-key',
    model: 'dall-e-3',          // 或 'dall-e-2'
    size: '1024x1024',           // 图片生成尺寸
    quality: 'standard'          // 或 'hd'
  };
</script>
💡 提示：如果已经在 Playground 中配置了 API Key 和 Base URL，<ai-canvas> 会自动从 localStorage 读取配置，无需重复设置。

💡 使用示例

基本用法

<ai-canvas p="一只穿着宇航服的猫在月球上漫步"></ai-canvas>

指定样式

<ai-canvas
  p="a watercolor painting of a Japanese garden in autumn"
  width="600"
  height="400"
  alt="Japanese Garden Watercolor"
></ai-canvas>

🎨 工作原理

1. 发送描述
将 p 属性中的描述发送到文生图 API 作为 prompt。
2. AI 绘制
文生图模型（如 DALL-E 3）根据描述生成图片，通常需要 10-30 秒。
3. 渲染展示
生成完成后，图片淡入显示。支持所有标准 <img> 属性。
