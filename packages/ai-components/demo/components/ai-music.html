<ai-music>

可用

AI 驱动的智能音乐播放组件——通过自然语言描述想听的音乐，自动搜索匹配曲目并播放。 内置精美播放器 UI，支持播放列表、进度条、播放控制等功能。如果配置了 LLM，还能根据 prompt 生成定制化的播放器界面。

⚙️ 属性（Attributes）

属性类型默认值说明
p string — 必填。自然语言描述想听的音乐。修改后自动重新搜索并播放。
theme "light" | "dark" "light" 播放器主题。dark 使用深色背景。
autoplay boolean — 设置后自动开始播放（需用户交互授权）。
loop boolean — 设置后当前曲目播完自动循环。
width string — 播放器宽度。
height string — 播放器高度。

💡 使用示例

基本用法

<ai-music p="播放一首轻松的爵士乐"></ai-music>

深色主题 + 自动播放

<ai-music
  p="play some lo-fi hip hop beats"
  theme="dark"
  autoplay
></ai-music>

循环播放

<ai-music
  p="安静的钢琴曲，适合工作时听"
  loop
></ai-music>

LLM 定制 UI（需配置 LLM）

<!-- 配置 LLM 后，播放器 UI 会根据 prompt 描述自动定制 -->
<ai-music
  p="复古黑胶唱片风格的古典音乐播放器"
></ai-music>

🎵 工作原理

1. 搜索音乐
使用 Jamendo API 根据 p 属性的描述搜索匹配的免费音乐曲目，返回最多 5 首结果。
2. 渲染播放器（双模式）
模式 A：LLM 定制 UI（需配置 LLM）— 将搜索结果和 prompt 传给 LLM，流式生成与音乐风格匹配的定制播放器界面。
模式 B：内置播放器（免费兜底）— 使用内置的精美播放器 UI，包含封面、曲目信息、进度条和播放控制。
3. 播放控制
支持播放/暂停、上一首/下一首、进度拖拽、循环播放。多首曲目时自动显示播放列表。
